## Design Enforcement Rules

### Mandatory Structure
- Every page uses `grid grid-cols-1 md:grid-cols-12 gap-6`
- Every page has header pattern: `h1` + description paragraph
- Every card uses: `card bg-base-100 shadow` + `card-body`

### Color Sparsity (90% Rule)
- 90% of UI is neutral: `bg-base-100`, `bg-base-200`, `text-base-content`
- Color only for: status badges, primary button (ONE per view), focus states
- FORBIDDEN: Large colored panels, multiple btn-primary, Tailwind colors

### Component Selection
- Status signals: `badge` or `indicator` (NOT alert)
- Expandable info: `collapse collapse-arrow` (default CLOSED)
- Grouped actions: `join` container
- Process steps: `steps` component
- Metrics: `stats` component

### Mobile Requirements
- Touch targets: 44px minimum (btn-md)
- Grid responsive: `grid-cols-1 md:grid-cols-12`
- Test viewport: 375px width

## Quick Reference

```
STRUCTURE:  grid grid-cols-1 md:grid-cols-12 gap-6
CARDS:      card bg-base-100 shadow > card-body > card-title
BUTTONS:    btn btn-primary (ONE) | btn-outline | btn-ghost
STATUS:     badge badge-success | badge-warning | badge-error
INPUT:      form-control > label > input input-bordered
COLLAPSE:   collapse collapse-arrow (default closed)
```
